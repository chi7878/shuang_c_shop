SHUANG C 電商網站
=====================
> 這是一份透過 Vue Cli 製作的練習電商網站
主要在銷售手機與手機周邊配件


## DEMO

[https://chi7878.github.io/shuang_c_shop/dist/#/](https://chi7878.github.io/shuang_c_shop/dist/#/)

## 網站頁面功能
### 前台頁面
* 商品展示(商品分類、單一商品介紹與推薦商品)
* 會員登入
* 購物車
* 結帳

### 後台頁面
* 管理者登入
* 產品與優惠券建立
* 商品修改資訊
* 訂單查詢

## 使用技術

 Vue  
 Vue Cli  
 Vue-Router  
 Bootstrap  
 Axios
 Google Fonts
